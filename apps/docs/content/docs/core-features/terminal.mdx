# Core Features - Terminal

This guide covers terminal-related features and commands available in fumadocs.

## Basic Terminal Commands

fumadocs provides several terminal commands to help you manage your documentation:

### Development Commands

Start the development server:

```bash
npm run dev
```

Build documentation for production:

```bash
npm run build
```

### Content Management

Generate new documentation pages:

```bash
fumadocs new page
```

Build static files:

```bash
fumadocs build
```

## Terminal Features

- **Auto-completion**: Tab completion for commands
- **Help system**: Built-in help for all commands
- **Error handling**: Clear error messages and suggestions
- **Cross-platform**: Works on Windows, macOS, and Linux

## Best Practices

- Use version control for your documentation
- Keep commands simple and readable
- Document complex command sequences
